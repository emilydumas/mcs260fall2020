# Design document
# Live coding example from MCS 260 Lecture 15 (2pm)
# Emily Dumas

Make a unary calculator.

Display a current amount (float).
Wait for a command.
Each command has the form
    NAME ARGUMENT
NAME is the operation, e.g. add, sub, mul, div
ARGUMENT is a float that the command applies to.

Unary = single argument to every command

0.0000
>add 5
5.0000
>mul 10
50.0000
>sub 1
49.000
>
